## About this project

This project develops methods to extract real-time social signals from
large-scale YouTube data, with a focus on mapping perceptions of violence
from video comments. Combining natural language processing with network and
temporal analysis, it asks whether platform conversation data can serve as an
early, location-sensitive indicator of real-world events and sentiment.
